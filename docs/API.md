# Documentação da API

A API está exposta via Swagger/Scalar para fácil consulta e teste.

## Endpoints Principais

### Menu
- `GET /api/menu`: Consulta o cardápio disponível.

### Orders
- `GET /api/orders`: Lista todos os pedidos.
- `GET /api/orders/{id}`: Consulta um pedido específico.
- `POST /api/orders`: Cria um novo pedido.
- `PUT /api/orders/{id}`: Atualiza um pedido existente.
- `DELETE /api/orders/{id}`: Remove um pedido.

## Como acessar
Ao rodar o projeto, acesse `/scalar/v1` para visualizar a documentação interativa.
